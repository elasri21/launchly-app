Launchly HTML Template – Setup & Customization Guide

Thank you for choosing Launchly! This guide will help you edit and customize your landing page quickly and easily.

📁 Folder Structure

launchly/
├── css/
│   └── rule.css
├── img/
│   └── images and icons
├── js/
│   └── main.js
├── index.html
└── instructions.html

🛠 How to Edit Text

Open index.html in a code editor. Locate the section you want to change and replace the placeholder text.

<h1>Supercharge Your Startup Launch</h1>

➡ Change this to your own title or message.

🎨 How to Change Colors & Fonts

Colors and fonts are managed via CSS variables in css/rule.css. You can edit them in the :root section at the top of the file:

:root {
  --color-primary: #00bfa6;
  --font-primary: 'Poppins', sans-serif;
}

🖼 How to Replace Images

Place your images in the img/ folder and update the src attributes in index.html like this:

<img src="img/your-image.png" alt="Description" />

🔗 Changing Navigation Links

Navigation items link to sections using ids. For example:

<a href="#features">Features</a>

Ensure your section has the corresponding id:

<section id="features">...</section>

📱 Responsive Design

Launchly is fully responsive and works on all screen sizes. You can test responsiveness using browser dev tools or real devices.

📩 Contact Form Behavior

This template includes a styled form, but you'll need to connect it to a service like Formspree, EmailJS, or your own backend to make it functional.

💡 Additional Tips

  • Use VS Code or Sublime Text for editing.
  • Preview your site by opening index.html in your browser.
  • Host your final project on a premium hosting provider (not free hosting).
